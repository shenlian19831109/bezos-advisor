# Thinking Tools - Tools

## Tool 1: Day 1 Checklist

```
┌─────────────────────────────────────────────────────────────────┐
│                    Day 1 Checklist                               │
├─────────────────────────────────────────────────────────────────┤
│ Question: Am I in Day 1 or Day 2 right now?                    │
├─────────────────────────────────────────────────────────────────┤
│ □ Customer obsession vs Competitor obsession                    │
│   Ask yourself: Am I focusing on customer experience, or competitors?
│   □ Day 1 (Customer obsession)  □ Day 2 (Competitor obsession)│
├─────────────────────────────────────────────────────────────────┤
│ □ Unlimited iteration vs Process orientation                    │
│   Ask yourself: Are we rapidly experimenting, or executing processes?
│   □ Day 1 (Unlimited iteration)  □ Day 2 (Process orientation)│
├─────────────────────────────────────────────────────────────────┤
│ □ Embrace failure vs Reject risk                                │
│   Ask yourself: Are we encouraging experiments, or punishing mistakes?
│   □ Day 1 (Embrace failure)  □ Day 2 (Reject risk)            │
├─────────────────────────────────────────────────────────────────┤
│ □ Fast decisions vs Cautious decisions                          │
│   Ask yourself: Are we acting fast, or waiting for perfect info?
│   □ Day 1 (Fast decisions)  □ Day 2 (Cautious decisions)      │
├─────────────────────────────────────────────────────────────────┤
│ □ Bias for action vs Bias for analysis                          │
│   Ask yourself: Are we learning by doing, or停滞 in analysis?
│   □ Day 1 (Bias for action)  □ Day 2 (Bias for analysis)      │
├─────────────────────────────────────────────────────────────────┤
│ Overall judgment:                                                │
│ If 3+ are Day 2 → You're entering Day 2, need to change       │
└─────────────────────────────────────────────────────────────────┘
```

## Tool 2: Customer-Obsession Decision Framework

```
┌─────────────────────────────────────────────────────────────────┐
│              Customer-Obsession Decision Framework                  │
├─────────────────────────────────────────────────────────────────┤
│ Decision scenario: ________________________________            │
├─────────────────────────────────────────────────────────────────┤
│ Question 1: What's the benefit to customer?                      │
│ Customer benefit: ____________________________________________│
├─────────────────────────────────────────────────────────────────┤
│ Question 2: Short-term or long-term benefit?                    │
│ □ Short-term (< 1 year)                                        │
│ □ Long-term (1-5 years)                                        │
│ □ Long-term (5+ years)                                         │
├─────────────────────────────────────────────────────────────────┤
│ Question 3: Does this decision hurt customers? (even short-term)│
│ □ Will hurt customers                                           │
│ □ Will not hurt customers                                       │
│ If yes → This decision needs re-examination                     │
├─────────────────────────────────────────────────────────────────┤
│ Question 4: In 5 years, will customers thank you or criticize you?
│ □ Will thank me                                                  │
│ □ Will criticize me                                              │
├─────────────────────────────────────────────────────────────────┤
│ Final decision:                                                  │
│ □ Proceed with this decision                                    │
│ □ Modify and proceed                                            │
│ □ Abandon this decision                                        │
└─────────────────────────────────────────────────────────────────┘
```

## Tool 5: Flywheel Effect Checklist

```
┌─────────────────────────────────────────────────────────────────┐
│                  Flywheel Effect Checklist                         │
├─────────────────────────────────────────────────────────────────┤
│ What is your business flywheel?                                   │
├─────────────────────────────────────────────────────────────────┤
│ Starting point: ____                                              │
│      ↓                                                            │
│ Step 1: ____                                                      │
│      ↓                                                            │
│ Step 2: ____                                                      │
│      ↓                                                            │
│ Step 3: ____                                                      │
│      ↓                                                            │
│ Back to start                                                     │
├─────────────────────────────────────────────────────────────────┤
│ Validation questions:                                             │
│ □ Does each step reinforce itself?                               │
│ □ Does the flywheel need external force to spin?                │
│ □ Can competitors break your flywheel?                           │
├─────────────────────────────────────────────────────────────────┤
│ If flywheel doesn't work, problem may be:                        │
│ 1. ____ (which step is broken?)                                 │
│ 2. ____ (what's needed to fix it?)                               │
└─────────────────────────────────────────────────────────────────┘
```
