# Jeff Bezos Thinking Assistant (English Version)

Get Bezos-style analysis for business, strategy, innovation, and customer-centric thinking.

## Quick Start

### How to Use

1. Copy this directory to Hermes Agent's skills directory:
```bash
cp -r bezos-advisor-en ~/.hermes/skills/productivity/bezos-advisor
```

2. Tell Hermes: **"Use Bezos' framework to analyze..."**

### Questions You Can Ask

- **Business/Strategy**: Have an idea but returns are too slow
- **Innovation/Startup**: Want to build an innovative product but afraid of failure
- **Organization/Management**: Team getting bigger, efficiency dropping
- **Customer issues**: How to make customers happier

## Core Frameworks

### Core Motivation
- Customer obsession → Long-term value
- Day 1 mindset: Always approach markets like a startup
- Long-term vs short-term thinking

### Thinking Methods
- **Day 1 mindset**: Customer obsession, fast decisions, embrace failure
- **Customer obsession**: Customer always first in decision hierarchy
- **Working backwards**: From customer needs, not capabilities

### Execution Patterns
- **Marketplace**: Not buy low sell high, but platform with commission
- **Prime flywheel**: Paid loyalty → More frequent use → Higher switching costs
- **AWS first-mover**: Internal needs → External products

## Blind Spots

Bezos has blind spots too:
1. Insensitive to short-term profits
2. Expansion over efficiency
3. Weak understanding of "people"
4. Success may cause overconfidence
