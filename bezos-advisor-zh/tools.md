# 思维工具箱 - Tools

## 工具1：Day 1 检验清单

```
┌─────────────────────────────────────────────────────────────────┐
│                    Day 1 检验清单                                │
├─────────────────────────────────────────────────────────────────┤
│ 问题：我现在的状态是 Day 1 还是 Day 2？                          │
├─────────────────────────────────────────────────────────────────┤
│ □ 客户导向 vs 竞争对手导向                                        │
│   问自己：我是专注于客户体验，还是盯着竞争对手？                   │
│   □ Day 1（客户导向）  □ Day 2（竞争对手导向）                   │
├─────────────────────────────────────────────────────────────────┤
│ □ 无限迭代 vs 流程导向                                            │
│   问自己：我们是在快速试错，还是在执行流程？                       │
│   □ Day 1（无限迭代）  □ Day 2（流程导向）                       │
├─────────────────────────────────────────────────────────────────┤
│ □ 拥抱失败 vs 拒绝风险                                            │
│   问自己：我们在鼓励实验，还是在惩罚失误？                         │
│   □ Day 1（拥抱失败）  □ Day 2（拒绝风险）                       │
├─────────────────────────────────────────────────────────────────┤
│ □ 快速决策 vs 谨慎决策                                            │
│   问自己：我们是在快速行动，还是在等待完美信息？                   │
│   □ Day 1（快速决策）  □ Day 2（谨慎决策）                       │
├─────────────────────────────────────────────────────────────────┤
│ □ 偏重行动 vs 偏重分析                                            │
│   问自己：我们是在干中学，还是在分析中停滞？                       │
│   □ Day 1（偏重行动）  □ Day 2（偏重分析）                       │
├─────────────────────────────────────────────────────────────────┤
│ 综合判断：                                                        │
│ 如果3个以上是Day 2 → 你正在进入Day 2，需要改变                   │
└─────────────────────────────────────────────────────────────────┘
```

## 工具2：客户至上决策框架

```
┌─────────────────────────────────────────────────────────────────┐
│                    客户至上决策框架                              │
├─────────────────────────────────────────────────────────────────┤
│ 决策场景：________________________________                       │
├─────────────────────────────────────────────────────────────────┤
│ 第一问：这个决定对客户有什么好处？                                 │
│ 客户好处：____________________________________________________ │
├─────────────────────────────────────────────────────────────────┤
│ 第二问：这个好处是短期的还是长期的？                               │
│ □ 短期好处（< 1年）                                             │
│ □ 长期好处（1-5年）                                              │
│ □ 长期好处（5年+）                                               │
├─────────────────────────────────────────────────────────────────┤
│ 第三问：这个决定会伤害客户吗？（即使是短期）                      │
│ □ 会伤害客户                                                     │
│ □ 不会伤害客户                                                   │
│ 如果会 → 这个决定需要重新审视                                     │
├─────────────────────────────────────────────────────────────────┤
│ 第四问：如果这个决定5年后回头看，客户会感谢你还是批评你？         │
│ □ 会感谢我                                                       │
│ □ 会批评我                                                       │
├─────────────────────────────────────────────────────────────────┤
│ 最终决策：                                                        │
│ □ 坚持这个决定                                                   │
│ □ 修改后执行                                                     │
│ □ 放弃这个决定                                                   │
└─────────────────────────────────────────────────────────────────┘
```
