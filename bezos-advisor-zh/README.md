# 贝索斯思维答疑助手 (中文版)

以贝索斯的逻辑和视角帮你分析商业、战略、创新、客户导向等问题。

## 快速开始

### 使用方法

1. 把这个目录复制到 Hermes Agent 的 skills 目录：
```bash
cp -r bezos-advisor-zh ~/.hermes/skills/productivity/bezos-advisor
```

2. 对 Hermes 说：**"用贝索斯的框架分析一下..."**

### 可以问的问题

- **商业/战略**：有个想法但回报太慢
- **创新/创业**：想做一个创新产品但怕失败
- **组织/管理**：团队越来越大效率越来越低
- **客户问题**：如何让客户更满意

## 核心框架

### 底层驱动力
- 客户至上 → 长期价值
- Day 1心态：永远用创业公司状态面对市场
- 长期主义 vs 短期主义

### 思维方法
- **Day 1心态**：客户导向、快速决策、拥抱失败
- **客户至上**：决策层级永远是客户第一
- **逆向工作法**：从客户需求倒推，不是从能力出发

### 执行模式
- **市场平台**：不是买进卖出，而是搭平台收佣金
- **Prime飞轮**：付费忠诚 → 更频繁使用 → 更高转换成本
- **AWS先发**：内部需求 → 外部产品

## 核心金句

```
"客户永远是第一，其他一切都会随之而来。"

"保持Day 1的心态。Day 2是停滞期，接着是衰落，
 然后是死亡。"

"在新世界里，你的毛利率就是你增长机会的地方。"
```

## 盲点

贝索斯也有盲点，提建议时注意：
1. 对短期利润不敏感
2. 扩张优先于效率
3. 对"人"的理解偏弱
4. 成功可能过度自信
