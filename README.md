## What is This?

Bezos Advisor is a skill that lets you consult with "Jeff Bezos" on any question. It uses his Day 1 mindset, customer obsession, and long-term thinking—not generic advice.

> **Version 2.0** — Significantly expanded from v1.0 with customer LTV framework, competition strategy, and complete Q&A scenarios.

---

## Core Frameworks

### Motivation
- **Customer obsession → Long-term value** — Amazon's mission is to be "Earth's most customer-centric company"
- **Day 1 mindset** — Always approach markets like a startup, never let big-company disease take over
- **Long-term thinking > Short-term gains** — Amazon's 37% annualized returns since 1997 vs 10% S&P 500

### Thinking Methods

| Framework | Description |
|-----------|-------------|
| **Day 1 vs Day 2** | Day 1 = customer obsession, rapid iteration, embracing failure. Day 2 = process disease, risk avoidance, competitor focus |
| **Customer LTV/CAC** | Customer Lifetime Value vs Customer Acquisition Cost. LTV/CAC > 3 = sustainable moat |
| **Working Backwards** | Start from customer needs, not company capabilities. Write PR/FAQ first, then build |
| **Margin = Growth** | Your margin is your opportunity for growth, not profit to distribute |

### Execution Patterns

| Pattern | Amazon Example |
|---------|----------------|
| **Marketplace Flywheel** | More sellers → More selection → Lower prices → More customers → More sellers |
| **Prime Flywheel** | Pay annual fee → Use more → More loyal → Spend 4x non-members |
| **AWS First-Mover** | Internal needs → External products. "If we solve our own problems, we can solve everyone else's" |

## 核心框架

### 底层驱动力
- **客户至上 → 长期价值** — 亚马逊使命："成为地球上最以客户为中心的公司"
- **Day 1心态** — 永远用创业公司状态面对市场，警惕大公司病
- **长期主义 > 短期利益** — 亚马逊1997年以来年化回报率37%，标普500同期10%

### 思维方法

| 框架 | 描述 |
|------|------|
| **Day 1 vs Day 2** | Day 1=客户导向、快速迭代、拥抱失败。Day 2=流程导向、风险规避、关注竞争对手 |
| **客户LTV/CAC** | 客户终身价值 vs 获取成本。LTV/CAC > 3 = 可持续护城河 |
| **逆向工作法** | 从客户需求出发，不是从能力出发。先写新闻稿/FAQ，再做产品 |
| **毛利率=增长** | 你的毛利率就是你的增长机会，不是要分掉的利润 |

### 执行模式

| 模式 | 亚马逊案例 |
|------|------------|
| **市场平台飞轮** | 更多卖家 → 更多选择 → 更低价格 → 更多客户 → 更多卖家 |
| **Prime飞轮** | 付年费 → 用得越多 → 越忠诚 → 消费是非会员4倍 |
| **AWS先发** | 内部需求 → 外部产品。"如果我们能解决自己的问题..." |
